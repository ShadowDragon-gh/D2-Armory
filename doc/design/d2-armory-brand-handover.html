D2 Armory — brand handover

Logo lockups, icon marks, wordmarks, and the full color token system for the D2 Armory loadout manager. Everything on this page is built from the same CSS custom properties listed below, so it can be dropped directly into the app's stylesheet.

Logo assets

Six variants: full lockup, icon only, and wordmark only — each with a solid background and a transparent background. Transparent versions are shown on a checkerboard so the transparency is visible; the actual files have no fill behind the artwork.

The icon-only mark is the one to use as an app icon / favicon — it reads clearly down to ~32px. Use the transparent versions on any non-surface-1 background; the solid-background versions are for contexts where you can't guarantee what's behind the logo (e.g. pasting into a doc or slide).

Color tokens

Steel-and-bronze dark theme. Surfaces step from near-black up through card and popover elevations; bronze is the single accent, reserved for interactive and brand moments so it doesn't get diluted.

These rarity tokens are named generically (common / uncommon / rare / legendary / exotic) rather than tied to any specific game's exact palette — safe to use as your own design system rather than a reproduction of Bungie's UI.

Typography

A condensed, geometric display face for headings and stat numbers (tactical/HUD feel), paired with a clean grotesk for body text and a monospace for stats or data tables. These are suggested Google Fonts pairings — not the literal font used in the logo artwork above, which is set in a generic bold sans for portability.

display / headings
Rajdhani · Oswald
body
Inter · Barlow
mono / stats
JetBrains Mono
